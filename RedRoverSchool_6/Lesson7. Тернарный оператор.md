## ТЕРНАРНЫЙ ОПЕРАТОР СОСТОИТ ИЗ 3 ОПЕРАТОРОВ: ##
### 1) Condition ###
> Это логическое выражение, которое может принимать истинное или ложное значение

### True value ###
> Значение, которое будет присвоено если в value будет true

### false value ###

```
x, y = 50, 55
s = x if x > y else y
print(s)
```

* РЕЗУЛЬТАТ: 55*

## НОВЫЙ ОПЕРАТОР ##

```
value = 5
match value:
    case 1:
        print(1)
    case 2:
        print(2)
    case 3:
        print(3)
    case 4:
        print(4)
    case 5:
        print(5)
```
