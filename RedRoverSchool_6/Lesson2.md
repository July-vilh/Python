* sep = '\n' разделяет строку + между символами можно передавать значение

* end = вставялет строку со значением которое мы передадим (В КОНЦЕ СТРОКИ)

Эти 2 вещи можно совмещать, например 

```
print(1, 2, 3, sep ='!', end = '?')
```
